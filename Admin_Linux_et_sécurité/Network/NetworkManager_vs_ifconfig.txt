====== NetworkManager vs ifconfig ======

===== Ifup/down =====
This is the legacy system for managing network connections.
It uses files stored inside ''/etc/network/interfaces'' folder.

===== Network Manager : the network management daemon =====
This is the default manager used by latest versions of Ubuntu.

''sudo service network-manager start''

**Profiles**
[[https://developer.gnome.org/NetworkManager/1.14/nm-settings.html|NetworkManager connection profiles]] are stored in ''/etc/NetworkManager/system-connections/'' folder.
They must be kept private, thats why each file must
* have permissions set with 600
* be owned by ''root:root''

**Front end**
* ''nmtui'' — Text User Interface for controlling NetworkManager
* ''network-manager-gnome''
