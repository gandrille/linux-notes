sed

Contents

    option -i pour modifier le fichier source

    \x... pour parler du caractère hexadécimal.

    Exemple, pour supprimer les carctres ^M en fin de ligne :
    sed -i "s/\0D$//" mon_fichier.txt

    Notons que ^M = 13, soit 0D en hexa…

    "s/…/…/g" pour remplacer toutes les occurences.

    substitute "foo" with "bar" ONLY for lines which contain "baz" : sed '/baz/s/foo/bar/g'

    Print the line before a line containing 'password' in the file 'config' : sed -n '/password/{x;p};h' config

    • affiche la troisième ligne sed -n '3p'
    • affiche la dernière ligne sed -n '$p'
    • affiche les lignes 3 à 8 sed -n '3,8p'
    • affiche à partir de la ligne 3 sed -n '3,$p'
    • affiche la ligne 5 puis 1/3 sed -n '5~3p'
    • supprime la première ligne sed '1d'

    • affiche les lignes qui contiennent un pattern sed -n '/PATTERN/p'
    • affiche les lignes qui NE contiennent PAS un pattern sed -n '/PATTERN/!p'
    • affiche les lignes à partir de pattern, jusqu'à la 8 sed -n '/PATTERN/,8p' regarder aussi sed -n '/<PATTERN/,+8p'
    • supprimer entre deux tokens sed "/token1/,/token2/d"
    • afficher entre deux tokens sed -n "/token1/,/token2/p"
