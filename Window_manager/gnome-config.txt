====== Configuration de Gnome ======

===== Pour suprimer les bookmarks =====
Editer le fichier ''~/.config/user-dirs.dirs'' et commenter les XDG

===== Ajouter la possibilité de créer un fichier vide =====
configurer le ''XDG_TEMPLATE_DIR'' de ''~/.config/user-dirs.dirs'' et créer un fichier vide dans le dossier template.

===== Ajouter open terminal here =====
voir contrib avec les screen shots :
http://askubuntu.com/questions/76712/setting-nautilus-open-terminal-to-launch-terminator-rather-than-gnome-terminal

ou sinon :
1. ''sudo apt-get install nautilus-open-terminal''
2. ''nautilus -q''

La seconde commande permet de faire un refresh de la config.

===== Créer un raccourci dans les applications =====
dans ''~/.local/share/applications''

créer un fichier ''.desktop'', comme par exemple, ''eclipse.desktop'' avec le contenu :

'''
[Desktop Entry]
Name=Eclipse
Comment=SensiNact Eclipse
Exec=/home/eg244698/eclipse/eclipses/studio/eclipse -data /home/eg244698/eclipse/workspaces/studio/workspace
StartupNotify=true
Terminal=false
Type=Application
Icon=/home/eg244698/eclipse/eclipses/studio/icon.xpm
'''


gsettings

ATTENTION : ''gconftool-2'' est l'ANCETRE (qui peine à disparaitre…) de ''gsettings''. Il faut donc utiliser ''gsettings'' en priorité.

Connaitre tous les schemas (catégories) disponibles : ''gsettings list-schemas''

Lister les clés dans un schéma : ''gsettings list-keys org.gnome.desktop.background''

Connaitre la valeur d'une clé

* ''gsettings get org.gnome.desktop.background picture-uri''
* ''gsettings get org.gnome.desktop.background picture-options''

Setter une clé : ''gsettings set org.gnome.desktop.background picture-uri 'file:///usr/share/backgrounds/gnome/FootFall.png'''
